🔐 JWT Spring Boot 3 Demo

Ứng dụng JWT Authentication đang chạy thành công!

📱 Trang Web:

Login Page Profile Page H2 Database

🔧 API Endpoints:

POST /auth/signup - Đăng ký user mới
POST /auth/login - Đăng nhập và nhận JWT token
GET /users/me - Lấy thông tin user (cần token)
GET /users/ - Danh sách tất cả users

✅ Tính năng hoàn thành:

  • ✅ JWT Authentication & Authorization
  • ✅ Spring Security Configuration
  • ✅ Exception Handling (GlobalExceptionHandler)
  • ✅ AJAX Frontend với jQuery
  • ✅ Thymeleaf Templates
  • ✅ H2 In-memory Database
  • ✅ CORS Configuration
🚀 Để test:
1. Click "Login Page" để đăng nhập qua giao diện
2. Hoặc sử dụng Postman/curl để test các API endpoints
3. H2 Console để xem database (URL: jdbc:h2:mem:testdb, User: SA)
